20200109 WC模拟考试 覆盖 - dp + 计数 + bitset

Description

求 $n$ 个节点的有相无环图使得最小点覆盖为 $k$ 的方案数，答案对 $2$ 取模。

Solution

首先将最小点覆盖的模型转化为最大独立集。

由于答案对 $2$ 取模，所以解决这个问题的关键是对称性。

考虑钦定两个点，若两个点的连边情况不同，那么显然我们可以将两个点调换，这样新的图由于形态相同，所以最小点覆盖数一样，这样在模 $2$ 意义下就抵消了，对答案没有贡献。

所以我们只需考虑两个点向外连边相同的情况。

首先，我们给每个点都赋上一个权值，最初全部为 $1$，并且转化为求带权最大独立集。

考虑若两个点 $x$ 之间没有连边，那么两个点要么同时选，要么同时不选，此时可以将两个点合并，并加入一个巨大点 $2x$。

若两个点 $x$ 之间有连边，则两个点只能选其一，所以去掉其中一个 $x$，对答案没有影响。

于是我们得到了转移：

设 $f[S]$ 表示点权的集合为 $S$ 时最大独立集为 $k$ 的图的个数。

那么
$$
f[S] \leftarrow f[S \backslash {x}] + f[S \backslash {x,x} \cup {2x}]
$$
若集合中有两个相同的权值，所以说我们每次有两种操作：

  1. 丢掉一个权值
  2. 将两个相同的权值 $x$ 合并为 $2x$

由于我们初始的点权全部为 $1$，所以最终我们得到的集合，也是 $dp$ 的边界情况，一定会是若干个互不相同的 $2$ 的正整数次幂，${2^{k_1},2^{k_2},\dots,2^{k_m}}$。

接下来是分类讨论，首先我们可以将 $k$ 二进制分解，且可以知道它的分解是唯一的。

也就是说 $S & k \ne k$，即 $k$ 不是 $S$ 的子集，那么肯定是不合法的情况。

容易知道 $S = k$ 时，方案数为 $1$。

再考虑 $k \subseteq S$ 时，$S$ 最多只能比 $k$ 多一个元素，因为若大于等于 $2$，那么这些点间两两连边或不连边最大独立集数不变，且一一对应，所以在模 $2$ 时对答案没有贡献。

否则，设 $S = k\cup { x }$，若 $x> \min k$，由于 $x$ 可以和比它小的那些边所以连边，所以方案数是 $2^w$，所以对答案贡献为 $0$。所以 $x < \min k$ 时，为了使 $x$ 不被选上，必须和那些权值比它的点连边，方案数为 $2^m-1$，所以对答案的贡献为 $1$。

由此我们得到结论，$f[S]=1$ 当且仅当 $S=k$ 或 $s = k + 2^x$，且 $2^{x+1} | k$。

所以我们可以设 $g[i][j]$ 表示 $i$ 个点权为 $1$ 的点，变成 $j$ 的二进制分解的方案数。

转移时枚举所有满足 $g[i-1][k]=1$ 的 $k$，考虑加入一个点权为 $1$ 的点后，可以不断的与地位连续的一段 $1$ 合并然后扔掉，这样就枚举出 $k$ 可能到达的状态。

最后这样复杂度是 $O(N^2)$ 的，加入 $bitset$ 后就玄学的过了。
