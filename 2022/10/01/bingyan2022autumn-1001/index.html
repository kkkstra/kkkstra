冰岩2022秋招实习记 - 1001

学习计划

  1. 熟悉 django 框架的基本构造以及操作
  2. 将 django 项目连接到 MySQL 数据库
  3. 实现阶段一的基础功能

项目进程

  1. 新建并初始化 django 项目 oidc
  2. 将项目 oidc 连接到 MySQL 数据库中
  3. 完成 user 应用中 User 类的构建，包括 id, username, password, email, intro 五个字段
  4. 完成 注册，登录 和 修改信息 三个基本操作的 api

总结

  1. django 的一个项目中包含了若干个应用，每个应用相对独立，可以迁移到不同项目中，例如项目 oidc 中包含 user 这个应用
  2. django 的基本核心是 urls.py, models.py 和 views.py

    urls.py 负责识别发送过来的 request，并把这些请求分派给 views.py 中相应的请求相应函数。而在 views.py 中函数的执行过程中，可能会调用到数据库的数据，这时候就会调用 models.py。

  3. 三个基本操作的 api 详见 bangumoe 的 README.md
