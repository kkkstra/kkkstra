语句

INSERT INTO

INSERT INTO 语句用于向表中插入一行或多行新记录。

SELECT

PostgreSQL SELECT 语句用于从数据库中选取数据，并将结果存储在一个结果表中，称为结果集。

LIMIT 子句

OFFSET 子句

从一个特定的偏移开始提取记录。

ORDER BY

对一列或者多列数据进行升序（ASC）或者降序（DESC）排列

GROUP BY

GROUP BY 语句和 SELECT 语句一起使用，用来对相同的数据进行分组。

GROUP BY 在一个 SELECT 语句中，放在 WHRER 子句的后面，ORDER BY 子句的前面。

应用示例

WITH 子句

WITH 子句有助于将复杂的大型查询分解为更简单的表单，便于阅读。这些语句通常称为通用表表达式（Common Table Express， CTE），也可以当做一个为查询而存在的临时表。

HAVING 子句

HAVING 子句筛选分组后的各组数据。

HAVING 子句必须放置于 GROUP BY 子句后面，ORDER BY 子句前面。

DISTINCT

DISTINCT 关键字与 SELECT 语句一起使用，用于去除重复记录，只获取唯一的记录。
